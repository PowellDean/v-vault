# Vault

## Introduction

Vault is a first attempt at creating simple wrapper for Hashicorp's excellent
Vault secrets system API written in [V](https://vlang.io).

It is intended to allow programmatic access to read wnd write secrets and
related metadata stored in a Vault instance. For more information on what Vault
is and how it can be used, visit the official Vault
[website](https://vaultproject.io)

## Pre-Requisites

* Hashicorp's executable for Vault, available from their website
* The latest V compiler (V 0.2.4 75830f1)

## General Usage

### Connecting to Vault

In this example, we use an authentication token generated by Vault to connect,
and we explicitly pass in the URL to the Vault server:

```v
module main
import client

fn main() {
x := client.new_client('http://127.0.0.1:8200', client.Authtype.token, 'root_token')
    println(x)
}
```

You can also preset the environment variable $VAULT\_ADDR, in which case there
is no need to pass the URL of the server:

```v
module main
import client

fn main() {
x := client.new_client('', client.Authtype.token, 'root_token')
    println(x)
}
```

Note that if you do not pass the Vault URL explicitly, _and_ the environment
variable $VAULT\_ADDR is not preset, the client will use the default
value 'http://127.0.0.1:8200'.

You can also authenticate to a Vault server with a username/password, provided
that option has been set by your Vault administrator. To log into Vault with
your username and password:

```v
module main
import vault

fn main() {
    cl := vault.new_client('', vault.Authtype.username, 'username', 'password')
    println(cl)
}
```

### Reading a secret

The API currently supports both Vault version 1 and version 2 secrets. To read
a version 1 secret:

```v
    s_v1 := cl.get_secret_v1('cubbyhole', 'foo') or {
        println(err)
        vault.Secret_v1{}
    }
    println(s_v1)
```

If the secret cubbyhole/foo is found, this will print output similar to this:

```
client.Secret_v1{
    request_id: '9e635499-73f3-a5c6-d628-d0dd737bfe8b'
    lease_id: ''
    renewable: false
    lease_duration: 0
    data: {'name': 'bar'}
}
```
